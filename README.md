# Git-Github
Repositório destinado a aprender sobre Git e GitHub

# Comandos Úteis:
<ul>
  <il>git remote add origin(apelido) url: conecta o diretório local ao repositório do git hub</il>
  <il>git remote remove origin: usada para trocar o repositório</il>
  <il>git remote set-url origin(apelido) nova_url: serve para trocar a url (caso esteja errada)</il>
  <il>git remote rename origin novo-origin: trocar o apelido do repositório</il>
  <il>git pull origin main: puxa os dados do repositório remoto para o local, evitando um conflito de dados e garantindo uma sincronização</il>
  <il>git log -> copiar o id e usar git revert id: reverter um commit -> git push origin main</il>
  <il>git reset --hard id: apaga o commit, só deve ser utilizado localmente</il>
  <il>git commit --amend -m "Mensagem": altera a mensagem do commit antes de ser enviada</il>
</ul>
